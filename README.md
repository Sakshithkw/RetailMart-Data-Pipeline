# 🛒 RetailMart Data Pipeline
### Junior Data Engineer Assignment — RetailMart Pvt. Ltd

---

## 📌 About This Project

RetailMart Pvt. Ltd. collects daily sales data from stores across India.  
The raw data is messy — it has **missing values**, **duplicate records**, and is spread across **multiple CSV files**.

This project builds a complete **ETL (Extract → Transform → Load) pipeline** that:
- Cleans and processes the raw data
- Merges it across files
- Loads it into a SQL database
- Generates business reports and insights

---

## 🗂️ Project Structure

```
RetailMart-Data-Pipeline/
```

## 🚀 How to Run

**Step 1 — Clone the repository**
```bash
git clone https://github.com/Sakshithkw/RetailMart-Data-Pipeline.git
cd RetailMart-Data-Pipeline
```

**Step 2 — Install dependencies**
```bash
pip install pandas numpy
```

**Step 3 — Run the pipeline**
```bash
python retailmart_pipeline.py
```

The script will automatically:
- Create the CSV files
- Clean and process the data
- Load everything into `retailmart.db`
- Print all reports in the terminal

---

## 🔄 Pipeline Steps

```
Load CSVs  →  Clean Data  →  Merge & Transform  →  Load to SQLite  →  Run Reports
```

| Task | What It Does |
|------|-------------|
| Task 1 — Data Ingestion | Load all 3 CSV files into pandas DataFrames, check shape and missing values |
| Task 2 — Data Cleaning | Remove duplicates, fill missing quantity with 0, drop rows with missing amount |
| Task 3 — Data Transformation | Merge all 3 DataFrames, calculate total_revenue, group by city |
| Task 4 — SQL Loading | Load final data into SQLite, query Top 3 best-selling products |
| Task 5 — Reporting | Revenue per store per day, summary report with key business metrics |
| Task 6 — Pipeline Function | Wrap all steps in run_pipeline() with full error handling |
